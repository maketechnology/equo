It's different

You don't have to recode your existing old Java desktop application to get a web-like one running in the Desktop. Not anymore. Equo not only provides the ability to create modern desktop applications from scrach, but it also allows you to migrate your old style application easily. It takes care of your code base, reusing it, so you can focus on the thing that matters, the UI.

All web applications can be easily migrated to the Desktop too. The code reuse that Equo provides is unprecedented.
